Course Contents & Slides

Key Topics

The course will cover the following key topics in Database Systems:

  • Introduction to RDBMS
  • Structured Query Language (SQL)
  • Relational Algebra & ER Model
  • Relational Database Design
  • Storage and File Structure
  • Application Development
  • Indexing and Hashing
  • Query Processing
  • Query Optimization
  • Transactions
  • Concurrency Control
  • Recovery Systems
  • Introduction to No-SQL Databases
